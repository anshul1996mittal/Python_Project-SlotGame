# Python Slot Game
This is a simple slot machine game made in Python. It runs in the terminal.

How it works
You deposit some money to start
Choose how many lines you want to bet on (1 to 3)
Enter bet amount per line
The slot spins and shows result
If symbols match in a line, you win

-Run the game
python slot_game.py

Notes
This is a basic project to practice Python concepts like loops, functions, and conditions.
